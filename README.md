## Installation

### TextMate
 - Download a .tmTheme file
 - Copy the file to ~/Library/Application Support/TextMate/Themes

----------
### Sublime Text
- Download a .tmTheme file
- Browse Packages from the Command Palette (Ctrl+Shift+P on Windows/Linux, ⇧⌘P on OS X)
- Copy the .tmTheme file to Packages
